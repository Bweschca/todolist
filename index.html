Konfiguracja sync

Aby mieć zadania na każdym urządzeniu, potrzebujesz darmowego klucza z JSONBin.io.

krok 01

Załóż konto na JSONBin

Wejdź na jsonbin.io i zarejestruj się (bezpłatne, bez karty)

Otwórz JSONBin.io →
krok 02

Wygeneruj klucz API

Po zalogowaniu: kliknij nazwę użytkownika → API Keys → Create Access Key → zaznacz wszystkie uprawnienia (READ, CREATE, UPDATE, DELETE) → skopiuj klucz

krok 03

Wklej klucz poniżej

Ten sam klucz wpisz na każdym urządzeniu – dane będą wspólne.
